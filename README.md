# CreditResolve

A credit resolution management dashboard for banking operations.

**Live App:** [https://creditapp-chi.vercel.app/](https://creditapp-chi.vercel.app/)

## Features

- Classified, Non-Accrual, Principal Recovery & Suspects tabs
- Sortable data tables with pagination
- Risk Unit Overview with interactive charts (collapsible)
- Add / Edit / Delete client records
- Comments per client with history
- Filters by responsibility centre, leader, associate, team lead

## Tech Stack

- [Next.js 16](https://nextjs.org) — App Router
- TypeScript
- Tailwind CSS v4
- Recharts
- Deployed on [Vercel](https://vercel.com)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.
